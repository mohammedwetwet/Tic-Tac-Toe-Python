# Tic-Tac-Toe-Python

A simple Tic-Tac-Toe game developed with Python to practice programming fundamentals, data structures and algorithms, object-oriented programming, and problem-solving skills.

## Features
- Two-player Tic-Tac-Toe game
- Simple and clean interface
- Built using Python
- Beginner-friendly project

## Skills Used
- Python Programming
- Object-Oriented Programming (OOP)
- Problem Solving
- Data Structures and Algorithms
